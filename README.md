# Guess the Word Game

A simple "Guess the Word" game built with HTML, CSS, and JavaScript.

## How to Play

1. Open the `index.html` file in your web browser.
2. A random word will be chosen, and you will see a series of underscores representing the letters in the word.
3. Click on the letter buttons to guess the letters in the word.
4. If you guess a correct letter, it will appear in the word.
5. If you guess all the letters correctly, you win!
6. Click the "New Word" button to start a new game with a new word.
