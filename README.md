# Guess the Word Game

A simple, grid-based "Guess the Word" game built with HTML, CSS, and JavaScript.

## How to Play

1. Open `index.html` in your web browser.
2. Start typing to guess the 5-letter word.
3. The letters will be colored based on your guess:
    - **Green:** Correct letter in the correct position.
    - **Yellow:** Correct letter in the wrong position.
    - **Red:** Incorrect letter.
4. You have 5 attempts to guess the word.
5. If you fail, the game is over, and the correct word is revealed.
